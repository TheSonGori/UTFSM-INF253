INTRUCCIONES

Version de Dr.Racket utilizada: version 8.6 [cs]

Se asume que siempre se entregaran valores correctos a las funciones, por ejemplo no se puede dar como
parametro #\A (o cualquier letra distinta de m o M) en las funciones de umbral, y que las funciones lambda
que se pasan por parametros siempre estan bien definidas.
Esto aplica para cada unas de las funciones.

Para hacer uso de las funcion eval tuve que ingresar:

	#lang scheme (current-namespace (make-base-namespace))

Ya que por lo menos en mi computador sin este, eval lo toma como error de funcion no definida.
	
Recursividad: -> Para las funciones donde nos piden recursividad simple, las funciones actuan como tal
			   en los ejemplos entregados en el PDF, comprobado con el Debug de DrRacket.
		    	-> Lo mismo para las funciones donde se pide recursividad de cola.
